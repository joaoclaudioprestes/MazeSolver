# Algoritmo DFS (Busca em Profundidade)

O DFS explora o labirinto indo o mais fundo possível antes de retroceder.

## Como funciona

1. Usa uma pilha (ou recursão) para explorar os caminhos.
2. Vai fundo em um caminho até não poder mais, então retrocede.
3. Pode não encontrar o caminho mais curto.

## Vantagens

- Simples de implementar.
- Útil para explorar todos os caminhos possíveis.

## Exemplo de uso

```python
path = dfs(maze, start, goal)
```
