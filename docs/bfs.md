# Algoritmo BFS (Busca em Largura)

O BFS explora o labirinto em camadas, garantindo encontrar o caminho mais curto em grafos não ponderados.

## Como funciona

1. Usa uma fila para armazenar os nós a serem explorados.
2. Explora todos os vizinhos de um nó antes de passar para o próximo nível.
3. Marca nós visitados para evitar ciclos.

## Vantagens

- Garante o caminho mais curto em grafos não ponderados.
- Simples de implementar.
